Comment ça marche ?

Putain de code ! est une communauté de développeurs passionés qui veulent partager leur savoir ainsi que leurs expériences.
Si vous êtes un développeur qui aime ce que vous faites, vous en faites déjà partie.

Nous aimons garder les choses simples. Dans ce sens, contribuer à ce projet devrait être facile.

Il y a une multitude de possibilités pour aider.
